# Data Analysis Sales Dashboard
this project focuses on analyzing sales data and building an interactive dashboard to understand performance across countries, cities, products, and time.  The data was cleaned and processed using Python and Excel, then visualized using Power BI to generate clear and actionable insights.

## 📌 Objective 
The goal of this project is to:
- clean and prepare raw sales data
- analyze patterns in sales performance
- identify top-performing products and locations
- present insights in a clear dashboard 

## ⚡️ Tools 
- Python (Pandas)
- Excel
- Power BI

## 📊 Dataset 
The dataset includes:
- order details
- product categories
- cities and countries
- dates (monthly data)
- cost and sales-related information
---
## 👩🏻‍💻 Project Workflow

### 1. Data Cleaning (Python & Excel)
- removed missing and inconsistent values  
- formatted columns and corrected data types  
- prepared data for analysis  

### 2. Data Processing (Pandas)
- grouped and aggregated data  
- calculated totals and counts  
- prepared structured data for visualization  

### 3. Data Visualization (Power BI)
Built an interactive dashboard that shows:

- total number of orders  
- number of countries and cities  
- sales distribution by country  
- orders by product category  
- monthly trends  
- top cities by number of orders
